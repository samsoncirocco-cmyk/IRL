# Architecture Validation Reports

This directory contains automated architecture validation reports generated by the IRL Architecture Validation Agent.

Reports are generated:
- Weekly on Mondays at 9 AM UTC
- On code changes to core IRL source files
- On manual workflow trigger

The latest report is always available at: `architecture-validation-latest.md`

## Report Naming Convention

`architecture-validation-[timestamp].md`

Example: `architecture-validation-1704067200000.md`

## Retention

- Local reports: Kept indefinitely (git-tracked)
- GitHub Actions artifacts: 90 days

## Usage

To view the latest validation report:

```bash
cat agents/reports/architecture-validation-latest.md
```

To list all reports:

```bash
ls -lht agents/reports/architecture-validation-*.md
```
